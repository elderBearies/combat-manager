Endpoints

GET:

/customMonsters: takes a limit parameter and returns a random selection of all of the monsters users have added. Defaults to 1 if no limit is given.

/searchMonsters: takes a term parameter and searches both the external api and the local api for the first monster that fits it.

POST:

/addMonster: Allows users to add monsters to the local api via a simple form.
